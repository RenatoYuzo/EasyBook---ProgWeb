# Descrição
Projeto para Trabalho de Programação Web, com a utilização do JSF no Tomcat 9.

## Dependencias:
  * Java EE 8
  * JSF 2.3
  * JSTL 1.2
  * Weld Servlet Shaded 3.1.1 (CDI 2.0)
  * Primefaces 7.0
 
## Ambiente de desenvolvimento:
  * Eclipse 2019-03
  * Tomcat 9.0.20
  * Banco de dados PostgreSQL e PgAdmin 4
